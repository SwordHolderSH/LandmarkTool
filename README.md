# LandmarkTool
A tool for labeling 68 face landmarks based on python-Qt5.

## Prerequisites
 
 * Linux or Windows
 * Qt5
 * Python 3

## Getting Started
1. Run ```main.py```. 
2. Open a face image.
<div align=center><img src="https://github.com/SwordHolderSH/LandmarkTool/blob/main/demo/1.jpg" width="1000" /></div>
 
3. Click button "Set landmark". 
<div align=center><img src="https://github.com/SwordHolderSH/LandmarkTool/blob/main/demo/2.png" width="800" /></div>

4. Drag right mouse to select key points, and hold left mouse to move key points.
<div align=center><img src="https://github.com/SwordHolderSH/LandmarkTool/blob/main/demo/3.png" width="800" /></div>
5. Click button "Save landmark" to save a ```.npy``` file. 
<div align=center><img src="https://github.com/SwordHolderSH/LandmarkTool/blob/main/demo/4.png" width="800" /></div>
<div align=center><img src="https://github.com/SwordHolderSH/LandmarkTool/blob/main/demo/save.png" width="600" /></div>
